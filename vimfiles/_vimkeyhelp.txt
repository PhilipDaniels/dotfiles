C-o           - bnext                  |*, #    - search for word under cursor
M-o           - tabnext                |b, e    - beg/end of word
C-p           - built-in Ctrl-P        |gg, G   - beg/end of file
M-p           - Ctrl-P plugin          |%       - matching paren
C-n           - toggle NERDTree        |g;      - last edited position
:e!           - revert file to saved   |
                                       |
F3, C-W <>    - adjust window width    |
F4, C-W -+    - adjust window height   |
z N           - set height to N        |
C-W s         - split window           |
C-W v         - vsplit window          |
C-W ]         - open tag in new window |
C-W f         - open file new window   |
C-W arrow     - move between windows   |
C-W C-W       - move between windows   |
C-W =         - make all equal size    |
C-W _         - maximize current       |
C-W r | R     - rotate windows         |
:[v]split file- split and load file    |
:hide         - hide current           |
:only         - hide all others        |
:b 2          - go to buffer 2         |
:ls           - list buffers           |
                                       |
================ TAGS =================|
C-]             - go to tag            |
C-W ]           - go to tag new window |
C-t             - return from tag      |
:tnext, tprev   - next/previous tag    |
:!ctags *.[ch]  - generate tags        |
:tags           - display tags stack   |
:tag name       - go to name           |
                                       |
================ TABS =================|
:tabedit [file] - open file in new tab |
:tabnew         - create a new tab     |
:tabclose!      - close current tab    |
:tabonly        - close all other tabs |
C-W gf          - edit file in new tab |



NERDTree
========
I               - show hidden files
r, R            - refresh (Root refresh
?               - toggle NERDTree help
o, CR           - open file
i               - open split
s               - open vsplit
t               - open tabsplit
o, O            - (recursively) toggle
m               - open menu (move, create, delete)
ESC             - exit the menu

Commands
========
:!cmd           - run external command
:read !cmd      - run external command and capture output
